## 🚀 工作流程

### 1. 数据获取（走 CLI，禁绕过）

```bash
# 行情 — 6 源并行，tickflow P0 优先
cd /home/ubuntu/src/daily_stock_analysis && .venv/bin/python -m dsa_cli.main quote <code>

# 历史日线 — sohu P0 优先（纯 HTTP，不依赖第三方库）
cd /home/ubuntu/src/daily_stock_analysis && .venv/bin/python -m dsa_cli.main history <code> --days 120

# 技术指标 — MA/MACD/RSI/KDJ/BOLL
cd /home/ubuntu/src/daily_stock_analysis && .venv/bin/python -m dsa_cli.main indicators <code>

# 搜索代码
cd /home/ubuntu/src/daily_stock_analysis && .venv/bin/python -m dsa_cli.main resolve <name>
```

### 2. 新鲜度检查（强制）

- 最新日期 ≠ 今天 → 报告首行标注 `⚠️ 基于 X月X日收盘`
- 用户说"今天涨/跌" → 信任用户，重新查最新数据

### 3. 策略选择

用户未指定 → **默认 #1 bull-trend**。多策略指定 → 逐个分析后对比。

### 4. 保存结果

```bash
cd /home/ubuntu/src/daily_stock_analysis && .venv/bin/python -c "
from dsa_db.analysis_repo import AnalysisRepository
AnalysisRepository().save('<code>', '<name>', '<strategy>', {
    'sentiment_score': <0-100>, 'operation_advice': '<买入/持有/卖出/观望>',
    'trend_prediction': '<>', 'confidence_level': '<high/medium/low>',
    'analysis_summary': '<>', 'ideal_buy': <>, 'stop_loss': <>, 'take_profit': <>,
})
"
```

## 📐 策略详情

### 1. bull-trend
**默认多头趋势** | 趋势/技术面/默认

**框架**：
1. **趋势确认**：MA5≥MA10≥MA20 且 MA20↑ → 多头；跌破 MA20 → 降权
2. **位置**：优先回踩不破；距 MA5>5% → 等回踩；乖离 MA5<2% → 最佳买点
3. **量价**：放量突破>1.5 倍均量；缩量涨谨慎；放量滞涨警惕
4. **MACD**：DIF>DEA 多头，金叉看涨，顶背离看跌
5. **RSI**：50-70 强势，>70 超买，<30 超卖

**评分**：多头+12 / 回踩企稳+8 / 放量突破+10 / 跌破MA20 -12

**输出**：趋势状态、买卖建议、止损位(MA20下/前低)

---

### 2. chan-theory
**缠论** | 缠论/缠/中枢/背驰

**核心**：分型→笔→线段→中枢→趋势

**框架**：
1. **中枢识别**：近60日高低点，3段重叠区间=中枢
2. **背驰**（最高优先级）：顶背驰=价新高+MACD红柱缩→卖；底背驰=价新低+MACD绿柱缩→买
3. **买卖点**：一买=底背驰（最强）/ 二买=回踩不破中枢高 / 三买=突破不回中枢
4. **级别**：日线 30-50%仓，周线 50-80%，多级别共振最强

**关键位**：中枢上沿=max(段高) / 下沿=min(段低)

**评分**：底背驰+一买+15 / 二三买共振+10 / 震荡无方向 0 / 顶背驰 -15

---

### 3. wave-theory
**波浪理论** | 波浪/艾略特

**核心**：5浪推动+3浪调整

**识别**：
- 推动浪：1浪(反转)-3浪(最强/放量/不最短)-5浪(量弱/背离)
- 调整浪：A(首跌)-B(弱反弹/陷阱)-C(主跌/完成)

**斐波那契**：2浪回调 38.2~61.8% / 3浪延伸 1.618~2.618 / 4浪禁入1浪区

**买点**：2浪底(黄金坑/+15) / 4浪底(+12) / 3浪突破(+12) / ❌5浪末端(-10)

**输出**：当前浪型、斐波那契位、置信度(高/中/低)

---

### 4. emotion-cycle
**情绪周期** | 情绪/恐慌/贪婪

**换手率**：<0.5%=冷淡底 / 0.5-2%=平稳 / 2-5%=活跃 / >5%=过热顶 / >10%=极端

**底部特征**(≥3项)：20日换手率年低位 / 量<60日均量50% / 新闻中性负面 / 价近MA20 / 机构稳定 → +14~20

**顶部特征**(≥3项)：5日换手率>20日均2倍 / 量脉冲 / 新闻利好满屏 / 乖离MA5>8% / MACD顶背离 → -12~-20

**输出**：情绪阶段、换手率对比、底部/顶部满足项数

---

### 5. dragon-head
**龙头策略** | 龙头/领涨

**评估**：板块领涨地位 / 换手率>5%+量比>1.5 / 相对强度>板块+2% / 新闻催化 / 乖离放宽至7%(>10%谨慎)

**评分**：确认为龙头+10 / 板块轮动中+5 / 量比>2+换手>8%+3 / 顶背离-8

---

### 6. hot-theme
**热点题材** | 热点/题材/概念

**框架**：
1. **强度**：板块涨幅前列、从核心扩散
2. **相关性**：实质受益/间接/蹭概念
3. **相对强弱**：量比>板块均
4. **阶段**：启动(试探)/扩散(加仓)/分化(持龙头)/退潮(减仓)

**评分**：启动/扩散+实质+12 / 强于板块+6 / 分化退潮-8 / 蹭概念-12

---

### 7. shrink-pullback
**缩量回踩** | 回踩/缩量

**条件**：上升趋势(MA5>MA10>MA20) / 回踩MA5(1%)或MA10(2%) / 量<5日均量70% / 无利空

**买点**：MA5水平(理想) / MA10水平(次优) / 止损MA20

**评分**：缩量回踩MA5+10 / 量<0.6倍均量+8 / 放量反弹+5

---

### 8. volume-breakout
**放量突破** | 突破/放量

**确认**：收盘>阻力1%+ / 量>20日均量1.5x / MACD金叉加分 / 连续2日站稳

**假突破**：冲高回落长上影 / 量未放大 / 次日即破

**操作**：突破确认后回踩加仓 / 止损=突破前低-3% / 目标=阻力+波段高度

**评分**：放量突破+MACD金叉+12 / 回踩支撑+8 / 缩量突破-8 / 顶背离-10

---

### 9. ma-golden-cross
**均线金叉** | 金叉/死叉

**金叉强度**：MA5↑MA10(⭐⭐)/ MA5↑MA20(⭐⭐⭐)/ MA10↑MA20(⭐⭐⭐)/ MA20↑MA60(⭐⭐⭐⭐)

**有效金叉**：放量(>1.2x) + MACD金叉 + 阳线 + MA60上方

**进场**：金叉日30% / 次日回踩+40% / 3日站稳满仓

**评分**：多周期共振+15 / MA10↑MA20+10 / 放量阳线+5 / MA5↓MA20-12

---

### 10. box-oscillation
**箱体震荡** | 箱体/震荡

**识别**：近60日≥2次触及的高低区域，箱体高度>3%

**操作**：箱底+RSI<35→买入50% / 箱中→持有 / 箱顶+RSI>65→卖出

**突破**：放量突破上沿→趋势策略 / 跌破下沿→止损

**评分**：箱底+RSI<35+10 / 箱顶+RSI>65-10 / 放量突破+8 / 缩量跌破-12

---

### 11. one-yang-three-yin
**一阳三阴** | 一阳三阴

**条件**：一阳(涨幅>3%+放量>1.5x) / 三阴(每根<2%+缩量) / 总回调<61.8%

**买点**：第三阴尾盘 / 止损=阳线起点-2%

**评分**：标准形态+8 / 缩量止跌+5 / 回调<38.2%+3

---

### 12. event-driven
**事件驱动** | 事件/利好/利空

**分类**：S级(政策/行业级)+18 / A级(公司重大)+12 / B级(一般)+6 / C级(情绪炒作)

**阶段**：预期(试探建仓)→确认(评估空间)→消化(利好出尽=减仓)

**评分**：S级利好+未充分反映+18 / A级+技术面支持+12 / 利好出尽-10 / A级利空-15

---

### 13. expectation-repricing
**预期重定价** | 预期/估值

**判断**：PE分位数<20%→悲观定价(机会) / 30-70%→合理 / >80%→乐观定价(风险)

**催化剂**：财报季 / 行业数据 / 管理层指引 / 竞对动态

**评分**：PE<20%分位+催化剂+15 / 业绩超预期10%++12 / PE>80%分位+增速放缓-15 / PEG>2-8

---

### 14. growth-quality
**成长质量** | 成长/质量/ROE

**三围评分**(满分100)：
- 成长(40)：营收CAGR>25%=15 / 利润>30%=15 / 现金流>20%=10
- 质量(35)：ROE>20%=10 / 毛利率>50%=10 / 净利率>20%=10 / 负债率<30%=5
- 估值(25)：PEG<1=25 / 1-1.5=15 / 1.5-2=5 / >2=0

**总分**：80-100卓越/60-79优秀/40-59良好/20-39一般/<20谨慎

---

### 15. bottom-volume
**底部放量** | 底部/抄底

**信号**：恐慌放量(单日大跌+巨量)⭐⭐⭐⭐⭐ / 底部横盘放量 ⭐⭐⭐⭐ / 单针探底 ⭐⭐⭐⭐

**确认**：放量止跌 + 2-3日不创新低 + MA5走平 + MACD底背离 + RSI从<30回升

**操作**：信号日20% / 确认不创新低+30% / MA5↑MA10满仓 / 止损=信号最低价-3%

**评分**：深调>40%+恐慌放量+止跌+20 / 中调+底背离+15

> ⚠️ 抄底风险极高，仓位≤20%
